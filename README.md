# Notas LCC
Notas que tome durante mi paso por la Licenciatura en Ciencias de la Computacion (FCEIA - UNR).

## Sobre el material
- Las notas fueron escritas con [Obsidian](https://obsidian.md/).
- Los PDF se compusieron a partir de varias notas individuales usando el plugin [Vault 2 Book](https://github.com/Mitra98t/vault2book-plugin).
- La exportacion a PDF se hizo con el plugin [Better PDF Export](https://github.com/l1xnan/obsidian-better-export-pdf).

Por ese proceso, es posible que haya errores de formato o de contenido.
